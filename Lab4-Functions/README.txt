
Quadratic Functions Lesson — Python Scripts
==========================================
This mini-pack shows core *function* concepts using the quadratic formula.

Files
-----
01_quadratic_basics.py
    Basic function definitions, docstrings, type hints, and returning multiple values.

02_named_args_and_defaults.py
    Named (keyword) arguments, keyword-only params, default values, and formatted output.

03_return_data_patterns.py
    Return patterns: tuple, dataclass, and dict—pros/cons of each.

04_exception_handling.py
    Custom exceptions, raising errors for invalid inputs, and try/except/else/finally.

05_user_input_validation.py
    Robust CLI input validation loop (type, finiteness, and domain checks).

Suggested Teaching Flow
-----------------------
1) Start with 01 to ground the math and function syntax.
2) Move to 02 to highlight named args, defaults, and keyword-only parameters.
3) Use 03 to discuss return-type design choices (tuple vs dataclass vs dict).
4) Use 04 to show defensive programming and graceful recovery.
5) Finish with 05 to practice real-world input validation and UX for CLI.

Run Examples
------------
python 01_quadratic_basics.py
python 02_named_args_and_defaults.py
python 03_return_data_patterns.py
python 04_exception_handling.py
python 05_user_input_validation.py
